Visual orchestration dashboard com React Flow — monitore e interaja com agentes de IA em tempo real.

Visao Geral

O Nexus Command Center e um dashboard de orquestracao visual que permite monitorar agentes de IA em tempo real. Cada agente e representado como um no no canvas, com conexoes animadas que mostram o fluxo de dados entre eles.

Views Disponiveis

Dashboard Canvas de Orquestracao

Vista principal com canvas visual, sidebar de navegacao, topbar com breadcrumbs e panel de metricas. Use para ter uma visao geral do sistema.

Workspace Workspace de Agentes

Vista expandida com canvas de agentes conectados, chat em tempo real, ticker de eventos e floating panels. Use para interagir diretamente com os agentes.

Navegacao

A navegacao esta disponivel de tres formas:

Sidebar

Barra lateral expansivel no lado esquerdo. Contem navegacao principal com icones e labels. Clique no botao de toggle para expandir/colapsar.

Command Palette

Pressione Ctrl+K ou Cmd+K para abrir a paleta de comandos. Digite para filtrar commands disponiveis:

  • Navegar para views
  • Acessar agentes
  • Abrir paineis
  • Acoes rapidas

Click direto

Clique em qualquer no no canvas para ver seus detalhes no NodeDetailPanel. Clique duas vezes para abrir o chat com o agente.
